Class Q.Ajax - extends Q.Class

This class creates objects that communicate using an XMLHttpRequest object.

Constructor

Q.Ajax ( options )
Parameters:
options <Object> the options object overrides default configuration. The type, contentType, url, and accepts properties are required. An object containing the options attributes. Members of this options object include:
  • string type: The request method.
  • string contentType: Request content type.
  • string accepts: Allowed response content types (separated by comma).
  • boolean cache: Use the browser's cache if available. "False" appends a random string to the url requested.
  • string url: Url to request.
