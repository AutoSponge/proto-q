multiton

void multiton ( name , defaults )
This class factory creates a subclass constructor with this instance as a __super__ property of the new class, inheriting this instance's prototype and configuration. The new class will use a multiton pattern (allowing global access to all instances). The new class constructor can be invoked like a function and will accept an options parameter which can override default configuration. To reference instances of a multiton, use the constructor.instance array. example: Q.Test.instance[0]
Parameters:
name <String> the reference to the new class
defaults <Object> the default configuration for new instances of the new class
Returns: void

singleton

void singleton ( name , defaults )
This class factory creates a subclass constructor with this instance as a __super__ property of the new class, inheriting this instance's prototype and configuration. The new class will use a singleton pattern (only allowing one instance to be created). The new class constructor can be invoked like a function and will accept an options parameter which can override default configuration. Singletons are not added to the request queue.
